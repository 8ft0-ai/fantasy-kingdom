# Export the Chronicle

Use this guide when you want a plain-text record of the current kingdom's history.

## Export from ANNALS

1. Open the left control drawer.
2. Choose **Export**.
3. Allow the browser to download the generated text file.

For seed `1234567`, the expected filename is:

```text
annals-1234567.txt
```

## Check the result

Open the downloaded file and confirm that it contains the Chronicle entries visible during the session.

The export is created entirely in the browser. It does not require a backend, account or browser persistence API.

Export after any Act or significant event you need included in the record.
